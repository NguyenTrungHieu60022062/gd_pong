#  Pong Game with Godot 4.3
Welcome to the Pong Game built with Godot 4.7
## Features
- **Pong Gameplay:** PvE action.
- **Score Tracking:** Display score.
- **Smooth Movement:** Responsive movement.
- **Screen Boundaries:** Ball bounces.
## Installation and Running the Game
### Prerequisites
Godot v4.7.
### How to Run
#### 1.Clone the repository
#### 2.Open the project in Godot
#### 3.Run the game
### Controls
* Move Up:```text Up Arrow ```
* Move Down:```text Down Arrow ```
